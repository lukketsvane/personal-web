# Futti Krutti

Futti Krutti er eit omfattande designprosjekt som utforskar fargar, typografi og ulike bruksområde for eit heilskapleg visuelt uttrykk.

## Typografisystem

Typografisystemet til Futti Krutti er basert på eit unikt glyph-design som kombinerer moderne og tradisjonelle element. Dette systemet gir prosjektet ein distinkt visuell identitet som er både lesbar og estetisk tiltalande.
